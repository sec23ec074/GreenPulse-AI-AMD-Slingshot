# 🌱 GreenPulse AI – AMD Slingshot Hackathon

GreenPulse AI is a full-stack Carbon Intelligence Platform built using FastAPI and React.

## Features
- Carbon footprint calculation
- Energy forecasting (Prophet)
- Regression-based ML recommendation engine
- Department leaderboard
- Policy-based carbon reduction estimation

## Tech Stack
Backend: FastAPI, Prophet, Scikit-learn  
Frontend: React.js  
Deployment Ready

## How to Run

Backend:
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

Frontend:
cd frontend
npm install
npm start
